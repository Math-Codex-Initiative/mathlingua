Fields

In the previous section, we describe what it means for a set to have an ordering of elements.

In this section we will describe sets with additional properties that allow binary operations to be applied to elements of the set.

The key algebraic structure we will be defining is a field since the real numbers turns out to be a field.

Now we can define what a field is. Essentially, a field is a set of elements with the operations of addition and multiplication that is an Abelian group under addition and an Abelian group (excluding $0$) under multplication.

Thus a field abstracts the familiar algebraic details of the real numbers. That is, the order of addition and multplication doesn't matter and any non-zero real number $x$ has a multiplicative inverse, namely $1/x$.

Now, to incorporate the details of the previous section, a field that has an order is called an ordered field.

The real numbers will turn out to be an ordered field with the ordering given by the familiar notion of $x < y$ for two real numbers $x$ and $y$.

Since an ordered field is a field, it contains the additive identiy, $0$. Further, since it is an ordered set, it has an ordering $<$. Thus, in any ordered field, one can describe what it means for an element of that field to be positive.

This is needed because the notion of a limit will be essential to describing the derivative of a function that is an essential part of the First Fundamental Theorem of Calculus.

Further to describe derivatives, it is necessary to describe the limit of a function, for which we will need to show that for a given $\epsilon > 0$ there exists a $\delta > 0$ such that the function satisfies certain properties.

This analysis thus depends on what it means for $x > 0$ for any real number $x$.
